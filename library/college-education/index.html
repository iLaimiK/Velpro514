# 小小说

清晨的阳光肆意挥洒在国立幽州大学的牌坊上，牌匾上 “顺天府大学堂” 几个大字在此时显得熠熠生辉。
三三两两的学生们不时出入校门，倒是让这个大门不那么冷清。学生们或大或小的交谈声与脚步声，为这里增添了一丝生活的气息。
林雨妍站在大门前，她身着休闲装，白色上衣与浅蓝色的裙子的搭配恰到好处；她娇小的身体配上姣好的面容，再加上一头过肩的黑色长发，放到众多女生中也是十分出众的存在。这使得不少学生经过她时都纷纷侧目，或赞叹，或称誉，让她觉得有些害羞。雨妍只是稍稍停下看看这昔日的大学门口，发觉自己有点引人注目后，夹紧了一下胳膊上放着的笔记本，匆匆地跟上进校学生们的步伐，快步走进校园。
门卫正悠闲地坐在椅子上看报纸，听到声音比以往要嘈杂一些，便抬眼看了一下，但似乎是把林雨妍当作学生了，只微微咳嗽两声，便继续看报纸去了。
雨妍走出一段距离后，学生也渐渐稀少了，现在她周围也清净了不少，除了时不时骑着自行车经过的学生，就只剩下戴着耳机漫步于道路两旁的人了。这里几乎没有人把注意力放在她身上，她才松了口气，开始观察起周围的环境来。很快，雨妍便根据脑海里的记忆辨认出所在的地方，往前走几步便找到了路旁的地图，确认无误后又对着地图熟悉了一遍校园的路线，随后便前往她所要去的目的地 —— 教学楼。
林雨妍走在路上，沿路的校园景色尽收眼底，她望着这带着七分熟悉的风景，不禁有些感慨，“物是人非啊～！” 她无意识地说出声来。几乎是同时，她用手掩住嘴，四处张望，发现没有人注意到她后，轻轻拍了拍心口，长吁一口气，以几乎只有自己能听见的声音说道：“幸好…… 果然还是会触景生情啊，还是赶紧过去那边吧……” 随后便步履匆匆地往教学楼走去。
到了教学楼，雨妍看了看时间，上课时间刚刚过半，她在窗外巡游过一楼的几间教室，发现教授们都正讲到兴致之处，虽然有些是在讲与课程无关的内容；而学生们不是在专心地做着自己的事情，就是在认真听课。雨妍不好意思中途打扰别人的课堂，便找了个空课室，坐在前排靠门的位置上，把笔记本摊开，拿起笔在上面写字，“国内大学教育未有任何长足进展，依然沿袭着以往的‘教学培养计划’进行教学，初步观察，现在大学课堂依旧不能顾及到每一个学生，且授课的老师会在教学过程中夹杂无关信息，其主要表现有……” 在写了满满一页纸后，她在末尾标上日期，并在开头空白处写上标题 “高等教育现状观察记录”。写完后，雨妍揉揉脑袋，看向手表时发现快要下课了，不过离下一节课开始还算早，但考虑到可能有人要用这个课室，她还是合上本子，收拾收拾后走出了空课室。
雨妍不紧不慢地穿梭于楼梯间，在到达她所要去的楼层时，下课铃声正好响起。她不慌不忙地走到走廊上，站到一旁不怎么显眼的地方，为下课后要赶去上另一节课的学生让出道路。雨妍看着急急忙忙的学生，听到嘈杂声中不时传出的 “点名”、“作业”、“抽查” 之类的话语，她心绪凌乱，结成了一张网，回忆起她上大学时的情况，与其别无二致。她摇了摇头，试图甩掉那些多余的念头，低下头，快速从逐渐稀疏的人群侧边走过。
教室里已经有几个学生坐着了，他们都低着头，勤奋地做着什么。雨妍的到来并没有引起谁的注意，她径直前往教室的后排，找了个位置坐下。这节课是她以前的教授负责的。她用左手撑着下巴，右手熟练地转起笔，思绪又飘到了别处。雨妍只在幽州大学待了一年，后来就去外国留学了，这一去就是八年多。如今回来这里，离踏上异国土地那时，已有十载。她回来是为了看看曾经的大学风光，也为了一睹这所国内顶尖学府的风采，还有就是…… 此时，上课铃声打断了雨妍的思绪，她回过神来，发现学生都已经坐在座位上了，教授也在讲台上准备好了。雨妍赶紧坐好，翻开笔记本，准备做些记录。
课堂并不算得上生动有趣，也没有到死气沉沉的程度，教授在讲理论部分时，会适时用点比喻，也会用自己或他人的经历来吸引学生的注意；讲到实践部分时会放些视频或者用板书来简化步骤，让学生尽量听得明白。但就总体而言，教授讲得都是教材上的内容，几乎没有补充内容，可以看出教授只是为了完成教学任务而走上讲台。一节课下来，雨妍听的都是过时的教学内容，她感到心底涌出了一股失望的苦水，将她全部的期待淹没其中。下课后，学生们熙熙攘攘地离去了，教授也手脚麻利地在收拾东西，而雨妍则是坐在那里，双眼无神地望着前方。教授在走出门前看了一眼雨妍，似乎是认出了她，张了张口，但看见她黯淡无光的眼神，心好像隐隐约约被扎了一针，无奈地叹了一口气，微微低着头离去了。
教室里静得出奇，连雨妍在笔记本上写画的声音都清晰可闻。“现在的信息时代，教师的主要作用是看管学生们学习，而非教导，倾囊相授的后果可能会导致学生自主学习能力呈现指数级上升。如今教学资源在互联网上唾手可得，教师如授人以渔，则相当于将自己的所有知识拱手让人。这些在中小学的低年级教师群体和家教中表现得特别明显，也是大部分教师‘授人以渔不如授人以鱼’的原因。在大学里，学生拥有完整成熟的自主学习能力，而且与老师毫无利益牵扯，此情况下，大学的教学活动更像是形式主义，上课对于老师和学生两方来说都显得毫无意义。而学校却有维持这一教学制度的必要性，它需要以此来培养学生的日常常规，保证学习进度，并在一定程度上对老师们进行约束。” 雨妍快速地书写着，娟秀的字体在纸上错落有致。写毕，她便快速地收拾好，在下一节课的学生到来前离开了。
午饭过后，雨妍走访了学校的图书馆、实验楼、文体楼等等各种校园内的设施，她意外地发现学校添置了不少文娱设施，不禁为科技发展带来的教育资源感到欣慰。但她却很惊讶图书馆即便是在上课时间也是挤得满满当当，实验楼无论什么时候都到处布满着匆匆忙忙的身影。虽然在一些地方还是能够感受到校园里青春活力的气息，但学生之间总是存在着一种微妙的气氛，说不明，道不清，感觉随时都会出现一股无形的压力，将学生们压得喘不过气来。
傍晚，雨妍走在学校人工湖旁的校道上，一页一页地翻着她带来的笔记本，她看着看着，感到有些糟心，便合上了本子，放回口袋里。环顾四周，学生们骑着自行车经过她的身旁，偶尔一两个走路的学生都是步履匆匆的；远处的图书馆灯火通明，每扇窗户都透露出淡淡的白光。夕阳的辉光斑驳地打在外墙上，与白光交相辉映。
雨妍失望地摇了摇头，倚靠着栏杆望向湖中心，平静的水面没有一丝波澜，如同死水一般，暗淡无光。她迷茫地望着湖面，莫名地有些感伤。
走出校门时，雨妍看着夕阳的最后一抹余晖渐渐消散在暮霭之中。她回头望去，大学牌坊上的几个大字也不再像白日那般闪耀……
